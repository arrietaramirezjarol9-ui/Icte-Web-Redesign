ICTE Web Redesign
📖 Descripción

ICTE Web Redesign es una propuesta de rediseño de la página web del Instituto Científico del Ejército (ICTE). El proyecto tiene como objetivo modernizar la presencia digital de la institución mediante una interfaz elegante, intuitiva y totalmente adaptable a diferentes dispositivos, priorizando la experiencia del usuario (UX), el rendimiento y la accesibilidad.

✨ Características principales
🎨 Diseño moderno con estilo Glassmorphism y estética tecnológica.
📱 Diseño Responsive para computadoras, tablets y teléfonos móviles.
⚡ Animaciones fluidas y transiciones interactivas.
🖼️ Galería de imágenes con Lightbox desarrollado en JavaScript puro.
📊 Contadores animados para estadísticas institucionales.
🚀 Navegación dinámica con una experiencia de usuario optimizada.
♿ Estructura semántica orientada a la accesibilidad y buenas prácticas SEO.
